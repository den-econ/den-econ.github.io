Data & Code

Replication packages and public datasets from our research

Replication Packages

All packages include data, code, and instructions to reproduce tables and figures in the corresponding paper. Code is primarily written in Stata and R.

  • Job Displacement and Long-Run Earnings Losses
    Replication package for Rivera & Vasquez (2024, AER). Includes cleaned administrative panel data (synthetic), Stata do-files, and R scripts for all figures. Restricted-access version of the underlying administrative data is available via application to [Data Authority].
    StataRPanel data
    GitHub ↗ Zenodo ↗ README
  • Microfinance and Female Entrepreneurship (Ghana RCT)
    Replication package for Chen & Osei (2024, JDE). Includes anonymized survey data from 1,200 households across 60 villages, Stata analysis code, and pre-analysis plan. Survey instrument also available.
    StataSurvey dataRCT
    GitHub ↗ AEA Data & Code Repository ↗ Pre-analysis plan ↗
  • Wage Dispersion and Search Frictions
    Replication package for Webb (2024, REStud). Includes simulated data matching moments from employer–employee panel, Julia estimation code, and bootstrapping scripts for standard errors.
    JuliaStructural estimation
    GitHub ↗ Zenodo ↗

Public Datasets

  • Local Labor Market Panel (LLMP), 2000–2022
    Annual county-level panel for the United States covering employment, wages, industry composition, and demographic characteristics. Constructed from the Quarterly Census of Employment and Wages (QCEW) and Census/ACS micro-data.
    Coverage: 3,000+ counties, 2000–2022. Format: CSV, Stata dta.
    US countiesAnnualLabor markets
    Download (CSV) Download (dta) Codebook GitHub ↗
  • School Quality Index, 2005–2022
    School-level quality index constructed from test scores, value-added estimates, and resource inputs for Colorado public schools. Used in Chen & Mendez (2025).
    Coverage: ~1,800 schools, 2005–2022. Format: CSV.
    Colorado schoolsEducation
    Download (CSV) Codebook

Software & Utilities

  • bunchR — R package for bunching estimation
    An R package implementing bunching estimators for kink and notch designs following Chetty et al. (2011) and Kleven (2016). Provides functions for estimating elasticities, constructing counterfactual density distributions, and inference via bootstrapping.
    RBunchingElasticity estimation
    GitHub ↗ Documentation ↗ CRAN ↗

Access to Restricted Data

Some of our research uses restricted-access administrative data that cannot be shared publicly. If you are interested in accessing similar data for your own research, we are happy to provide guidance on the application process. Please contact us.
